🏗️ 아키텍처

Motoo 프로젝트의 시스템 아키텍처, 데이터베이스 설계, 기술 스택 등 프로젝트의 전체적인 구조를 파악할 수 있습니다.

📚 개요

시스템 아키텍처 개요

  • 마이크로서비스 아키텍처 기반 설계
  • REST API와 메시지 큐를 통한 서비스 간 통신
  • 확장성과 안정성을 고려한 구조
  • 현대적인 기술 스택 활용

📊 시스템 아키텍처

Presentation Layer

API Gateway Load Balancer CDN

Business Layer

User Service Stock Service Order Service Portfolio Service Transaction Service Token Service Execution Service

Data Layer

PostgreSQL Redis Kafka AWS S3

🗄️ 데이터베이스 ERD

데이터베이스 설계

PostgreSQL을 기반으로 한 정규화된 데이터베이스 설계와 Redis를 활용한 캐싱 전략을 구현했습니다.

  • 정규화된 테이블 구조
  • 인덱스 최적화
  • 트랜잭션 관리
  • 백업 및 복구 전략

📈 확장성 전략
