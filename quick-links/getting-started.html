🚀 빠른 시작

1. 프로젝트 클론

git clone https://github.com/your-org/motoo.git
cd motoo

2. 환경 설정

# 환경 변수 설정
cp .env.example .env
# 데이터베이스 설정 수정
# Redis 설정 수정

3. 데이터베이스 설정

# PostgreSQL 실행
docker run -d --name postgres \
  -e POSTGRES_DB=motoo \
  -e POSTGRES_USER=motoo \
  -e POSTGRES_PASSWORD=password \
  -p 5432:5432 postgres:14

4. 애플리케이션 실행

# 백엔드 실행
./gradlew bootRun

# 프론트엔드 실행
cd frontend
npm install
npm start

🔍 자주 묻는 질문

Q: Motoo는 어떤 기술 스택을 사용하나요?

A: Spring Boot (Java), React (TypeScript), PostgreSQL, Redis 등을 사용합니다. 자세한 내용은 기술 스택 페이지를 참고하세요.

Q: 개발 환경 설정이 어려운데 도움을 받을 수 있나요?

A: 네, 지원 요청 페이지를 통해 개발팀에 문의하실 수 있습니다.

Q: 프로젝트에 기여하고 싶은데 어떻게 해야 하나요?

A: GitHub 저장소의 CONTRIBUTING.md 파일을 참고하거나, 개발팀 연락처로 문의하세요.

Q: 문서가 업데이트되면 어떻게 알 수 있나요?

A: 변경 사항 페이지에서 최신 업데이트를 확인할 수 있습니다.
